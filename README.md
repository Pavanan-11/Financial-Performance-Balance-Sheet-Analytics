# Financial Performance & Balance Sheet Analytics Dashboard 

This project delivers a complete financial reporting and balance sheet analytics solution developed in Power BI. 
It transforms raw accounting data (Trial Balance + Chart of Accounts) into an automated, insight-driven dashboard that supports variance analysis, profitability assessment, and financial statement evaluation.

# Objectives

Build an end-to-end financial model integrating Trial Balance with a structured Chart of Accounts.
Automate period-over-period variance for Assets, Liabilities, and Equity.
Provide insights into revenue patterns, expense behaviour, and financial health.
Build an interactive Power BI dashboard for executive decision-making.

# Key Analytical Insights

Revenue and Cost of Sales demonstrate a stable operating cycle with minimal volatility.
Gross margins remain consistent, indicating controlled operational efficiency.
Marketing expenses show periodic spikes, directly impacting monthly profitability.
Asset movements reflect working capital changes, while liability shifts highlight short-term operational funding needs.
Overall indicators point to financial stability and predictable expense behaviour.
